#Django Skeleton

##A startup repo giving the basic requirements to get you website up and running.

###Installation
    git clone git://github.com/joshfinnie/django-skeleton.git /YOUR-APP
    cd YOUR-APP
    virtualenv venv
    source venv/bin/activate
    pip install -r requirements.txt
